Tìm Hiểu Các Thẻ HTML Cơ Bản

Các thẻ kiểu none:

html, head, title, meta, link, script, style.

Các thẻ kiểu block level:

body, h1, div, p, center, ul + li, form + input, br, hr, table, iframe, blockquote, tt + code pre.

Các thẻ kiểu inline:

span, a, img, b, i, u, s, sub.

So sánh các kiểu tag none, tag block level, inline
