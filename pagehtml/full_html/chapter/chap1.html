Hôm nay mình cùng các bạn tìm hiểu những khái niệm cơ bản nhé.

1. WWW là gì ?

World Wide Web gọi tắt là Web hoặc WWW, mản lưới toàn cầu là một không gian thông tin toàn cầu mà mọi người có thể truy cập (đọc và viết) thông tin qua các thiết bị kết nối với mạng internet ...

Đọc thêm.

2. Web page là gì ?

Web page là một tài liệu (thương là HTML) được thiết kế để phân phối trên môi trường web.

3. Web site là gì ?

Web site là tập hợp các web page có liên quan đến 1 công ty hay có nhân

4. Domain name ?

Domain name là tên xác định web site
VD: www.leanhvuwebpage.com

5. DNS ?

DNS (Domain name service): một chương trình chạy trên server, chuyển tên miền sang IP và ngược lại

6. Server ?

Server chưa dữ liệu, tài nguyên và dịch vụ cho phép máy khác có thể khai thác và truy cập.

  • Một máy chủ có thể dùng cho một hay nhiều mục đích.
  • Tên máy chủ thường đuộc gắn với mục đính sử dụng
    • VD: Web server, File server, Mail server ...

7. Client ?

Client là máy tính dùng để kết nối và khai thác các tài nguyên trên máy tính chủ

  • Việc kết nối client với server và việc khai thác dịch vu của server tạo nên mô hình Client/Server
  • Một máy tính vừa có thể là serfrver vừa là client.

8. Trang web tĩnh ?

Trang web tĩnh chứa nội dung cố định (thường là HTML để cập nhật nội dung phải cập nhật trực tiếp trên HTML).

Không cho phép sử dụng tương tác, cập nhập dữ liệu trên trang web.

Một trang web chứa các hình ảnh chuyển động cũng có thể là trang web tĩnh!

9. Trang web động ?

Kết hợp HTML và mã lệnh.

Mã được thực thi trực tiếp trên server, gửi kết quả là HTML về người dùng.

Có khả năng tương tác với người sử dụng!

10. HTML là gì vì sao ta học ?

HTML : HyperText Markup Language - ngôn ngữ đánh dấu siêu văn bản.

Do Tim Berner-Lee phát minh và trở thành ngôn ngữ chuẩn để tạo trang web.

HTML dùng các thẻ (tags) để định dạng dữ liệu

Tạo khung/bản cho trang web.
