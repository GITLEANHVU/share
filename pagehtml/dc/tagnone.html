Tìm Hiểu Các Kiểu Thẻ None

Khối thẻ none không hiển thị nội dung bên trong

html, head, title, meta, link, script, style:

  1. html: thẻ mở đầu của trang HTML
  2. head: thẻ chứa các thẻ trong phần đầu của trang HTML
  3. title: tiêu đề của trang web
  4. meta: mô tả tổng quát về nội dung của trang web
  5. link: dùng để nhúng một tập tin nào đó vào trang web
  6. script: dùng để nhúng các tập tin JavaScript
  7. style: dùng để bao bọc nội dung về css
